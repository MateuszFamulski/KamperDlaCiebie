## Overview

Kamper Dla Ciebie is a cozy, adventure-themed website for camper van rental services in Poland. Built with the **Restaurantly** Bootstrap template as a foundation, the site has been adapted with warmer colors, camper travel content, bilingual support, and a modern gallery and contact form.

This version includes:

- Modern responsive design
- Polish and English language support
- Hero section with video overlay
- About & Why Us sections with animated cards
- Camper Offer details (bed, kitchen, shower, etc.)
- Benefits tabs (freedom, independence, flexibility)
- Thematic event slider (family, couple, friends)
- Interactive gallery using Glightbox
- Contact form with custom mailto functionality

## Getting Started

### Installation

1. Clone or download the repository.
2. Make sure to keep the folder structure as-is (`assets/`, `index.html`, etc.).
3. Open `index.html` or `index-en.html` in your browser.

### Customization

You can update your contact email and phone in:

```html
<a href="mailto:kamperdlaciebie@wp.pl">kamperdlaciebie@wp.pl</a>
<a href="tel:+48663709646">+48 663 709 646</a>
```

To replace gallery images, update the files in:

```
assets/img/gallery/
```

To change the video in the Hero section, replace:

```
assets/video/main-video.mp4
```
